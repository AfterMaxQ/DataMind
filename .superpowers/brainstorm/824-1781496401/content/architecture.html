Key Innovation: The Script IS the Index

What this unlocks

  • AI always knows how any dataset was created — just read the linked script
  • New sessions auto-load lineage graph → full historical context
  • data.describe gives instant schema understanding without reading entire files
  • Reproducibility: re-run the script chain to recreate any processed dataset

Questions to resolve

  • What happens when a user manually edits data outside the system?
  • How do we handle branching? (one raw → multiple processed variants)
  • Versioning: if a script is re-run with changes, do we overwrite or fork?
